📋 Tentang Proyek

Anjania Resto adalah aplikasi manajemen restoran berbasis web yang dirancang untuk membantu pengelolaan restoran secara digital dan terintegrasi. Sistem ini mendukung proses operasional mulai dari pemesanan pelanggan, pengelolaan meja, manajemen stok bahan baku, hingga laporan keuangan.

🚀 Instalasi

1. Clone Repository

git clone https://github.com/username/Anjania-Resto.git
cd Anjania-Resto

2. Install Dependency

composer install

3. Import Database

Import file SQL yang tersedia pada folder database.

4. Konfigurasi Database

Sesuaikan konfigurasi database pada file pengaturan aplikasi.

5. Jalankan Server

Gunakan Apache/XAMPP/Laragon atau web server yang mendukung PHP.
